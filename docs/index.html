使用方法

简单四步，开启智能文档处理之旅

1

配置 AI 提供商

在设置页面选择 AI 提供商，输入 API Key。支持 DeepSeek、Moonshot、智谱、百炼等多种选择。

2

上传文档

点击上传按钮，选择 Word 或 PDF 文档。系统自动提取文本内容，并进行向量化处理。

3

创建翻译任务

选择需要翻译的文档，点击"创建任务"。系统将自动翻译为英文，并生成中英双语 PDF。

4

智能问答

选择已向量化的文档，在聊天框输入问题。系统将基于文档内容进行智能问答，精准回答您的问题。
